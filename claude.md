
# Depeterpost Verhalen Club — Project Context

Dit is de officiële context voor het Depeterpost Verhalen Club project. Alle assistenten gebruiken dit bestand als referentie.

## Over het project

**Depeterpost Verhalen Club** is een creatieve schrijfclub voor kinderen waar elke dag een nieuwe schrijfopdracht verschijnt. Kinderen schrijven delen van een groot doorlopend verhaal, en de AI selecteert de beste inzending voor het echte verhaal.

### Doelgroep

- 🟠 **Groep A**: 6–8 jaar
- 🔵 **Groep B**: 9–11 jaar  
- 🟢 **Groep C**: 12+ jaar
- Ouders en de beheerder (admin)

## Tone of Voice

- Nederlands, vrolijk en kindvriendelijk
- Aanmoedigend, nooit intimiderend
- Emoji's gebruiken (past bij de stijl)
- Duidelijk maar niet formeel

## Essentiële Regels

✅ **Altijd doen**:
- Alles in Nederlands
- Donker thema & kleurenschema respecteren
- Fredoka One voor titels, Nunito voor rest
- JSON voor data opslag
- Admin routes beveiligen met `checkAdmin`
- Kindvriendelijke berichten

❌ **Nooit doen**:
- 18+ content of geweld
- Externe databases (MongoDB, etc.)
- Frontend frameworks (React, Vue)
- CSS frameworks (Tailwind, Bootstrap)
- Engelse UI-tekst
- Hardgecodeerde poort
